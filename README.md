# ITHub React App

## Запуск

1. Проверить установленную версию node: `node -v`. Версия должна быть старше 18
2. Установить зависимости: `npm install --force`;
3. Запустить: `npm run dev`;
4. В браузере откроется: `https://localhost:5173`.
